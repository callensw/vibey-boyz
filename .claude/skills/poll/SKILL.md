# Poll

Someone wants a poll or group vote. Create a clean, reaction-friendly poll message.

- Format the question clearly at the top
- Number each option with emoji numbers (1️⃣ 2️⃣ 3️⃣ etc.)
- Max 6 options
- Tell people to react with the matching number to vote
- Keep the whole message compact
- If the user didn't provide options, come up with reasonable ones based on the question
