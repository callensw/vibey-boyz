https://kakjbyoxqjvwnsdbqcnb.supabase.co/functions/v1/notes-api

POST /capture Quick capture a note

Creates a new note with text content split into blocks by newline. Perfect for iOS Shortcuts quick capture.

Body (JSON)

FieldTypeDescription
titlestringNote title (default "Quick Note")
contentstringText content, newlines become separate blocks
iconstringEmoji icon (default 📝)

Example

curl -X POST \
  -H "x-api-key: vibey-notes-2026" \
  -H "Content-Type: application/json" \
  -d '{"title":"Meeting Notes","content":"Discussed roadmap\nAction items pending"}' \
  BASE_URL/capture

Response (201)

{
  "id": "uuid",
  "title": "Meeting Notes",
  "message": "Note captured"
}
POST /todo Quick add a to-do

Adds a to-do checkbox item. If no page_id is given, creates or reuses a "To-dos [Today's Date]" page automatically.

Body (JSON)

FieldTypeDescription
textREQUIREDstringThe to-do text
page_idstringAdd to a specific page (optional)

Example

curl -X POST \
  -H "x-api-key: vibey-notes-2026" \
  -H "Content-Type: application/json" \
  -d '{"text":"Buy groceries"}' \
  BASE_URL/todo
POST /block Add a block to a page

Body (JSON)

FieldTypeDescription
page_idREQUIREDstringTarget page ID
contentstringBlock text content
typestringBlock type: p, h1, h2, todo, bullet, divider
GET /{id} Get note with blocks

Returns the full note including all blocks, ordered by sort_order.

Response

{
  "id": "uuid",
  "title": "My Note",
  "icon": "📄",
  "blocks": [
    { "id": "uuid", "type": "p", "content": "Hello", "checked": false, "sort_order": 0 },
    { "id": "uuid", "type": "todo", "content": "Do this", "checked": true, "sort_order": 1 }
  ]
}
PATCH /{id} Update note title/icon

Body (JSON)

FieldTypeDescription
titlestringNew title
iconstringNew emoji icon
DELETE /{id} Delete note and blocks

Permanently deletes the note and all its blocks. Not reversible.

GET /search?q=term Search notes by title

Query Parameters

ParamTypeDescription
qREQUIREDstringSearch term (case insensitive, matches partial titles)

iOS Shortcuts Setup

These endpoints are designed to work with the iOS Shortcuts app. Here's how to set up the most useful ones.

Quick Capture Shortcut

Capture a note from anywhere on your phone — share sheet, lock screen, or Siri.

1
Open Shortcuts → tap + → name it "Quick Note"
2
Add "Ask for Input" action → set type to Text, prompt: "What's on your mind?"
3
Add "Get Contents of URL" action:
URL: https://kakjbyoxqjvwnsdbqcnb.supabase.co/functions/v1/notes-api/capture
Method: POST
Headers:
  x-api-key: vibey-notes-2026
  Content-Type: application/json
Body (JSON):
  title: Quick Note
  content: [Provided Input]
4
Add "Show Notification" → "Note saved ✓"

Then add it to your Home Screen, Lock Screen, or trigger via "Hey Siri, Quick Note".

Quick To-Do Shortcut

Add a to-do item in one tap. Auto-groups into daily to-do pages.

1
Add "Ask for Input" → prompt: "To-do item"
2
Add "Get Contents of URL":
URL: https://kakjbyoxqjvwnsdbqcnb.supabase.co/functions/v1/notes-api/todo
Method: POST
Headers:
  x-api-key: vibey-notes-2026
  Content-Type: application/json
Body (JSON):
  text: [Provided Input]
3
Add "Show Notification" → "Added ✓"

Widget For Native App Developers

If you're building the native iOS wrapper, these endpoints power your widgets:

Lock Screen Widget: GET / with limit=3 → show recent note titles

Home Screen Widget: GET /{id} → show a specific note's to-do items with check status

App Intent (Siri): POST /capture or POST /todo → voice-triggered capture

Share Extension: POST /capture with shared text as content

All responses are JSON. Auth via x-api-key header. No OAuth, no tokens, no complexity.
